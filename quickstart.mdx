Once installed, invoke the skill in Claude Code by typing `/imessage-mail` followed by your intent in natural language. The skill classifies your request and routes it to the right agent automatically.

## Your first commands

<Tabs>
  <Tab title="Recent messages">
    Ask who has been in touch recently:

    ```
    /imessage-mail who texted me
    ```

    Example output:

    ```
    ## Recent iMessages

    | Contact              | Last Message        | Messages     | Preview                         |
    |----------------------|---------------------|--------------|---------------------------------|
    | Sarah Chen           | 2026-03-16 14:30    | 45 (↓23 ↑22) | "Hey, are we still on for..."   |
    | Mike Johnson         | 2026-03-16 12:15    | 12 (↓8 ↑4)   | "Thanks for sending that!"      |
    | +1555000111          | 2026-03-15 09:00    | 3 (↓3 ↑0)    | [Attachment: image]             |

    ## Recent Emails (5 unread)

    | Status | From                         | Subject               | Received      |
    |--------|------------------------------|-----------------------|---------------|
    | UNREAD | Sarah Chen (sarah@corp.com)  | Re: Project Update    | Mar 16, 18:30 |
    | Read   | GitHub (noreply@github.com)  | [repo] PR #42 merged  | Mar 16, 17:00 |
    ```
  </Tab>
  <Tab title="Read a conversation">
    Read full message history with a contact:

    ```
    /imessage-mail texts with Sarah
    ```

    Example output:

    ```
    ## Conversation with Sarah Chen

    ### iMessages (+15551234567)
    *25 most recent*

    **14:30 Me**: Hey, are we still on for dinner tonight?
    **14:32 Sarah**: Yeah! What time works?
    **14:33 Me**: How about 7?
    **14:35 Sarah**: Perfect, see you then!
    ```
  </Tab>
  <Tab title="Send a message">
    Send an iMessage with a natural language instruction:

    ```
    /imessage-mail text Sarah saying I'll be 10 minutes late
    ```

    The skill always shows you what it's about to send and asks for confirmation:

    ```
    **Ready to send:**

    **Type:** iMessage
    **To:** Sarah Chen (+15551234567)

    > I'll be 10 minutes late

    Send this? (y/n)
    ```

    Type `y` to send or `n` to cancel.
  </Tab>
  <Tab title="Search messages">
    Search across all iMessages and emails simultaneously:

    ```
    /imessage-mail search "invoice"
    ```

    Example output:

    ```
    ## Search Results for "invoice"

    ### iMessages — 2 matches
    **Mike Johnson (+15559876543)**
    - **Mar 14, 12:00 — Mike**: "Can you send me the invoice?"
    - **Mar 14, 12:05 — Me**: "Just emailed it over"

    ### Emails — 3 matches
    | Status | From         | Subject               | Received      |
    |--------|--------------|-----------------------|---------------|
    | Read   | QuickBooks   | Invoice #1042 paid    | Mar 15, 10:00 |
    | Read   | Mike Johnson | Re: Invoice request   | Mar 14, 12:30 |
    | UNREAD | Vendor Inc.  | Invoice #8837 due     | Mar 12, 09:00 |
    ```
  </Tab>
</Tabs>

## Common commands

| Command | What it does |
|---------|-------------|
| `/imessage-mail recent` | Recent iMessages and emails overview |
| `/imessage-mail messages from Sarah` | Full conversation with a contact |
| `/imessage-mail search "project deadline"` | Search across all messages and email |
| `/imessage-mail email` | Browse Apple Mail inbox |
| `/imessage-mail send a text to Mike saying I'm running late` | Send an iMessage |
| `/imessage-mail find attachments from David` | Find files shared in messages |

<Note>
You don't need exact command syntax — the skill understands natural language. "Who emailed me today?", "Show me texts from Mom", and "Did anyone send me a PDF?" all work.
</Note>
